# Immersive Interactive Portfolio Website

This website is designed as an interactive portfolio platform to showcase videos and highlight a personal brand, styled in urban streetwear culture with manga/anime-inspired visuals. Visitors will navigate through a Tokyo-style house, explore rooms, and interact with objects that reveal video content.

---

## Project Features:

### 1. Entry Point (Home Page):
- **Japanese Tokyo-style house** with a BMW E30 parked outside.
  - Clicking on the car reveals a cinematic video.
  - Visitors enter the house by clicking the door.

### 2. Living Room:
- Interactive features and an ambient design.
- Clickable objects, e.g., a TV showcasing the latest campaign video.
- Stairs leading to 3 rooms:
  - **Room (Personal):** Includes a Rottweiler dog that plays a video when clicked.
  - **Studio:** Posters and sound equipment with behind-the-scenes videos.
  - **Gaming Room:** Monitors displaying clips of content.

### 3. Page Navigation:
- Users navigate by clicking doors, triggering smooth transitions to the next room.

### 4. Visual Style:
- **Streetwear + Manga/Anime.**
  - Inspired by Bleach with bold typography, vivid colors, and manga-like transitions.
  - Dropdown menus styled with graffiti/street art.

### 5. Pop-Up Notice:
- On entering the homepage, users see a video pop-up of the latest release/campaign.

---

## Development

### Tech Stack:
- **Framework:** React.js for interactivity.
- **3D Modeling:** Three.js with Blender assets.
- **UI Prototyping:** Figma (manga styled visuals).
- **Animations:** GSAP (GreenSock Animation System) or Lottie Files.

### Get Started

1. Clone this repository:
```bash
 git clone https://github.com/itzmakeout/00.git
```
2. Install dependencies:
```bash
cd 00
npm install
```
3. Run the project locally:
```bash
npm start
```
4. Update the assets, videos, and styling for customization.
